# 🛒 Amazon Clone

A frontend Amazon clone built using HTML, and CSS 
This project replicates the core UI of Amazon including a responsive navbar, hero section, product grid, and footer.

---

## 🌐 Live Demo
👉 https://mehakrathoure.github.io/Amazon-clone/

---

## 🚀 Features

- 🔹 Responsive Navbar with search bar and cart  
- 🔹 Hero banner section  
- 🔹 Product grid layout with images and details  
- 🔹 Footer with multiple panels  
- 🔹 Clean and user-friendly UI  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3
